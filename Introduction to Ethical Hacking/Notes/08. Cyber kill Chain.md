# Cyber Kill Chain

## 📌 Introduction
The **Cyber Kill Chain** is a cybersecurity framework developed by Lockheed Martin to describe the stages of a cyberattack. It helps security professionals understand attacker behavior, detect threats early, and stop attacks before damage occurs.

By analyzing each phase, organizations can strengthen their defenses and improve incident response.

---

# 🔐 What is Cyber Kill Chain?

The Cyber Kill Chain breaks down a cyberattack into a series of steps that attackers commonly follow to compromise a target system or network.

It is useful for:
- Threat detection  
- Security monitoring  
- Incident response  
- Risk assessment  
- Defensive strategy planning  

---

# ⚔️ 7 Phases of Cyber Kill Chain

## 1️⃣ Reconnaissance

### 📖 Definition:
The attacker gathers information about the target.

### 📌 Examples:
- Collecting employee emails  
- Finding IP addresses  
- Social media research  
- Domain and DNS lookup  

### 🛠️ Techniques:
- OSINT  
- Google Dorking  
- Whois Lookup  

---

## 2️⃣ Weaponization

### 📖 Definition:
The attacker creates a malicious payload to exploit the target.

### 📌 Examples:
- Malware creation  
- Infected documents  
- Backdoor payloads  
- Trojanized software  

### 🛠️ Tools:
- Exploit kits  
- Malware builders  

---

## 3️⃣ Delivery

### 📖 Definition:
The attacker sends the weaponized payload to the victim.

### 📌 Methods:
- Phishing emails  
- Malicious links  
- USB devices  
- Fake software downloads  

---

## 4️⃣ Exploitation

### 📖 Definition:
The malicious file or vulnerability is triggered.

### 📌 Examples:
- Opening infected attachment  
- Clicking exploit link  
- Running malicious program  

### 🛠️ Techniques:
- Buffer Overflow  
- Code Execution  
- Macro Exploits  

---

## 5️⃣ Installation

### 📖 Definition:
Malware is installed on the victim system.

### 📌 Examples:
- Remote Access Trojan (RAT)  
- Spyware  
- Keylogger  

---

## 6️⃣ Command and Control (C2)

### 📖 Definition:
The compromised system connects to the attacker’s server for remote control.

### 📌 Purpose:
- Receive commands  
- Send stolen data  
- Download additional malware  

---

## 7️⃣ Actions on Objectives

### 📖 Definition:
The attacker performs the final mission after gaining control.

### 📌 Examples:
- Data theft  
- Ransomware encryption  
- System destruction  
- Espionage  

---

# 📊 Summary Table

| Phase | Purpose |
|------|---------|
| Reconnaissance | Gather target information |
| Weaponization | Create attack payload |
| Delivery | Send payload to victim |
| Exploitation | Trigger vulnerability |
| Installation | Install malware |
| Command & Control | Remote communication |
| Actions on Objectives | Complete attacker goal |

---

# 🛡️ Importance of Cyber Kill Chain

- Detect attacks early  
- Break attacks before completion  
- Improve SOC monitoring  
- Strengthen incident response  
- Understand attacker tactics  

---

# 🚫 How to Defend Against Kill Chain Attacks

- Security awareness training  
- Email filtering  
- Patch management  
- Antivirus / EDR solutions  
- Network monitoring  
- Firewall protection  
- Multi-factor authentication  

---

# 🎯 Conclusion

The Cyber Kill Chain provides a clear model of how cyberattacks happen step by step. By understanding each phase, defenders can stop attackers before they reach their final objective.
