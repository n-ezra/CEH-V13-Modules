# 🔍 Google Dorking (Practical)

## 📌 Overview
Google Dorking is a technique used in footprinting to extract sensitive or hidden information from search engines using advanced search operators.

---

## 🎯 Objective
- To identify publicly available sensitive data
- To understand how attackers use search engines for reconnaissance

---

## 🛠️ Tools Used
- Google Search Engine
- Browser (Chrome / Firefox)

---

## 📚 Common Google Dorks

| Dork | Description |
|------|------------|
| `site:example.com` | Search within a specific website |
| `filetype:pdf` | Find specific file types |
| `intitle:"index of"` | Find open directories |
| `inurl:admin` | Find admin pages |
| `intext:"username"` | Search for specific text |

---

## ⚡ Practical Execution

### 🔹 1. Finding Login Pages
